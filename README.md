# Basic AV
Hello, this is a basic AV that works by checking for signatures.

## Usage
1. Get this repo
   git clone https://github.com/filip-skwarczewski/BasicAV.git
2. Move into repo folder
   cd BasicAV
3. Run the main.go
   go run main.go --file=<fileToCheck>
4. Add more signatures
This only has 1 signature and a test signature
